Data Structures and Algorithms Questions

Below is a compiled list of Data Structures and Algorithms interview questions that will help you conquer that upcoming interview of yours!

  1. How do you find the largest and smallest number in an unsorted integer array?

  2. Given two binary trees, check if they are the same or not.

  3. Given a BST, convert it into a sorted linked list. Return the head of LL.

  4. Write a Python program to determine if a number is Prime or not.

  5. How to Reverse a Linked List ?

  6. Find all occurance of an element in an array.

  7. Given an array containing n distinct numbers taken from 0, 1, 2, ..., n. Find the missing number.

  8. Given an array of integers where each integer appears exactly once except for one integer which appears twice, find the duplicate number.

  9. Find the contiguous subarray within an array that has the largest sum.

  10. Given a collection of intervals, merge all overlapping intervals.

  11. Given an array of integers, return indices of the two numbers such that they add up to a specific target.

  12. Given an array nums of n integers where n > 1, return an array output such that output[i] is equal to the product of all the elements of nums except nums[i].

  13. Given an array prices where prices[i] is the price of a given stock on the i-th day, find the maximum profit you can achieve.

  14. Given an array of integers, 1 ≤ a[i] ≤ n (n = size of array), some elements appear twice and others appear once. Find all the elements that appear twice.

  15. Given an unsorted array of integers, find the length of the longest consecutive elements sequence.

  16. Given an array, rotate the array to the right by k steps, where k is non-negative.
